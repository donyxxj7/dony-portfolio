## 📂 Projetos em Destaque

### 1. 🔗 LinkHub: Agregador de Links Pessoal (Full-stack)

<a href="https://linkhub-0xw2.onrender.com"><img src="https://github.com/donyxxj7/dony-portfolio/blob/main/assets/LinkHub.png?raw=true" alt="Screenshot do Projeto LinkHub" width="700"></a>

Uma aplicação full-stack completa, inspirada no Linktree, que permite criar e gerenciar uma página de links personalizada.

* **Destaques:** Construído do zero com back-end em Node.js/Express, banco de dados SQLite, e um painel de administração seguro com autenticação.
* **Links:**
    * [**Ver Site Ao Vivo**](https://linkhub-0xw2.onrender.com)
    * [**Ver Código no GitHub**](https://github.com/donyxxj7/linkhub-fullstack-portfolio)

### 2. 🎬 Cine-Info: Buscador de Filmes (Front-end)

<a href="https://dashing-kataifi-8b89dc.netlify.app/"><img src="https://github.com/donyxxj7/dony-portfolio/blob/main/assets/Cine-Info%20ScreenShot.png?raw=true" alt="Screenshot do Projeto Cine-Info" width="700"></a>

Uma aplicação web interativa que consome a API do TMDB para permitir que usuários descubram, pesquisem e vejam detalhes sobre filmes.

* **Destaques:** Auditado com notas excelentes no Lighthouse (90+ em Performance e 100 em Acessibilidade), design responsivo e paginação.
* **Links:**
    * [**Ver Site Ao Vivo**](https://infocine.netlify.app/)
    * [**Ver Código no GitHub**](https://github.com/donyxxj7/cine-info)
